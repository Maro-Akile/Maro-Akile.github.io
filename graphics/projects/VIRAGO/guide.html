VIRAGO

Last updated: April 17, 2024 by Maro Akile


What is VIRAGO?

VIRAGO is a plugin for making VRoid models easier to use in the Blender software.


Requirements for the VIRAGO plugin:

Blender 4.1

VIRAGO is NOT a standalone program. It was developed to assist the Blender program. VIRAGO was developed with Blender 4.1 in mind, so it is currently unknown if older versions of Blender support VIRAGO. Feel free to test it on older Blender versions.

VRM Importer/Exporter

Without the VRM Importer/Exporter plugin, the VIRAGO project would never have existed. This plugin makes it possible to take .vrm files and place them in Blender. However, it loses a VRoid’s original control rigs and physics.

Wiggle 2

Wiggle 2 carries all the hair and cloth physics of VIRAGO’s processed VRoids. VIRAGO uses Wiggle 2 data to create hair and cloth physics.


Current Issues with VIRAGO:

> VIRAGO’s compatibility with Mac and Linux is currently unknown.

> The “Control Rig” operation freezes for ~5 seconds, but does not freeze the app. Performing other functions during this period may cause strange behavior. For best results, wait 10 seconds after pressing “Control Rig” before pressing anything else.

> The “Preset Actions” operation currently does not check for valid situations. Unless you are bugtesting VIRAGO, please only press Preset Actions after a successful Control Rig operation
